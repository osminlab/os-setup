# How I backup my PC

I tried the built-in tools that Microsft provied, but there are bullshit. So, for my convinece exist a free Software that help me with this, it's call [EaseUS Todo Backup Free](https://www.easeus.com/backup-software/tb-free.html).

For that software i use the following configuration:
1)  I  have two disk. One for OS and other for multiuses.
2) Backup once a week differencial.


Also i use Google Drive for my backup online, for save my personal data and WSL data.
